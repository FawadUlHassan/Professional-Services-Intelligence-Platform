# Business Glossary

## Purpose

This glossary defines the business meaning of key metrics and terms used in the Professional Services Operations & Profitability Intelligence Platform.

## Core Business Terms

| Term | Definition |
|---|---|
| Project | A customer-facing professional services engagement, solution delivery, or managed services requirement |
| Revenue | Customer-facing project value or expected billing amount |
| NTL Cost | Internal cost incurred by Nayatel for procurement, delivery, or execution |
| Profit | Revenue minus NTL Cost |
| Gross Margin % | Profit divided by Revenue |
| Markup % | Profit divided by NTL Cost |
| Existing Margin (%) | Current spreadsheet margin field; treated as raw markup percentage unless validated otherwise |
| Salesperson | Commercial owner responsible for sales coordination or lead handling |
| Project Engineer | Technical/delivery owner responsible for project execution |
| Project Sector | Customer industry or business segment |
| Requirement Type | Commercial source/type such as NTL own lead, tender, or RFQ/RFP |
| Progress Status | Current lifecycle state of the project |
| Expected Man Hours | Planned effort required for project delivery |
| Actual Man Hours | Actual effort recorded after project completion |
| Special Discount | Discount amount given to customer |
| OEM | Original equipment manufacturer or vendor brand |
| Distributor | Supplier or procurement channel |
| Payment Mode | Payment terms or commercial payment arrangement |
| Passive Work | Physical/passive infrastructure work involved in delivery |
| Equipment Provided | Equipment, product, or solution delivered to customer |

## Metric Formulas

```text
Profit = Revenue - NTL Cost

Gross Margin % = Profit / Revenue * 100

Markup % = Profit / NTL Cost * 100

Effort Variance Hours = Actual Man Hours - Expected Man Hours

Effort Variance % = Effort Variance Hours / Expected Man Hours * 100
